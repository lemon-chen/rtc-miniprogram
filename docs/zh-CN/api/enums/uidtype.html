Enumeration UidType

用户 ID 的类型。

note
  • 如果设置为整数，则建议的设置范围是 0 到 (232-1)
  • 使用字符串作为用户 ID 支持与 Native SDK 2.8 及以上版本或 Web SDK 互通。请确保 Native SDK 使用 User Account 加入频道。详见如何使用 String 型的用户名。
  • 一个频道内的所有用户必须使用同样类型的 uid，即必须都为整数或都为字符串。
